# RainFocus UI Challenge — Nate Valline

## Viewing the Build

The compiled static build is in the `out/` folder. Follow these steps to view it:

1. Unzip the project folder if you haven't already.

2. Open a terminal (Mac) or PowerShell (Windows).

3. Navigate into the project folder. Replace the path below with wherever you unzipped the folder:
   ```bash
   cd /path/to/nate-valline-rf-ui
   ```
   **Example on Mac:** `cd ~/Downloads/nate-valline-rf-ui`
   
   **Example on Windows:** `cd C:\Users\YourName\Downloads\nate-valline-rf-ui`

4. Run the following command:
   ```bash
   npx serve out
   ```

5. Open `http://localhost:3000` in your browser.

## Running the Development Server

**Prerequisites:** Node.js 18 or higher

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.
