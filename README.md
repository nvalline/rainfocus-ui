# RainFocus UI Challenge — Nate Valline

## Viewing the Build

The compiled static build is in the `out/` folder. To view it, open `out/index.html` directly in a browser, or serve it locally to avoid any asset path issues:

```bash
npx serve out
```

Then open `http://localhost:3000` in your browser.

## Running the Development Server

**Prerequisites:** Node.js 18 or higher

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.
